\# Smart AI Resume Analyzer



An AI-powered Resume Analyzer built using Python, Streamlit, OCR, NLP, and HuggingFace Transformers.



\## Features



\- Resume Upload (PDF/DOCX/Image)

\- OCR Text Extraction

\- AI Resume Summary

\- ATS Score Calculation

\- Skill Detection

\- Missing Skills Analysis

\- Role Prediction

\- Interview Question Generation

\- Interactive Visualizations

\- Downloadable PDF Report



\## Tech Stack



\- Python

\- Streamlit

\- HuggingFace Transformers

\- Tesseract OCR

\- Plotly

\- PyPDF2

\- ReportLab

streamlit run app.py
